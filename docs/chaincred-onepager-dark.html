ChainCred's Approach

Index what wallets do, not what they hold. Score actions across 6 EVM chains, apply multiplicative sybil penalties, and publish results as verifiable EAS attestations.

1. Expertise Score (0–1000) — weighted composite of 5 onchain signal categories
2. 7 Badges — specialist attestations (Builder, Governor, OG, Explorer, etc.)
3. Sybil Resistance — 8 heuristics that discount bots, not ban them
4. Onchain Proof — weekly Merkle root + EAS attestations for trustless verification

Why It's Built This Way

Actions Over Holdings

Token balances reward wealth. ChainCred only indexes transactions — deployments, votes, protocol interactions. You can't buy a score.

Multi-Signal Categories

Each category uses 4–9 capped signals. Over-capping prevents any single signal from dominating — no wallet can max all signals.

Multiplicative Sybil Penalties

Penalties compound: 3 flags = 0.6 × 0.7 × 0.4 = 0.168 confidence. Each extra flag bites exponentially.

Bear Market = High Signal

Activity during downturns is the hardest signal to fake retroactively. Cross-cycle persistence is near-unforgeable proof of genuine participation.

Trustless Verification

Weekly Merkle roots onchain let anyone verify a score without trusting ChainCred. Combined with EAS attestations — composable by any protocol.

Fail-Open Everything

If Etherscan, RPC, or Redis is down, scoring still works. Enrichments degrade gracefully — no external dependency blocks the core pipeline.
